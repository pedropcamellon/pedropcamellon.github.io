## Hi 👋, I am Pedro!

- AI/ML Engineer based in Miami
- Platform engineer on Azure — Container Apps, Temporal, Terraform
- Building [Folium EHR](https://github.com/pedropcamellon/folium) — open-source, production-grade
- Interested in durable execution, observability, and RAG systems
- AZ-104 in progress, DevOps Engineer cert next

At work, I build AI-powered tools on Azure — platform migrations, durable workflow orchestration, and production observability from the ground up. I've led migrations from Azure Functions to Temporal workers on Azure Container Apps, built Grafana dashboards with Prometheus and Azure Monitor, and I enjoy supporting teammates through PR reviews, onboarding, and architecture decisions.

I'm drawn to high-impact industries. I've worked in supply chain finance, healthcare, and now construction tech — domains where good engineering makes a real difference.

**What I work with:**

Python (FastAPI) - TypeScript/React - Azure (Container Apps, Cosmos, Monitor, Entra ID, DevOps Pipelines) - Temporal - Terraform - Docker - Prometheus - Grafana - GitHub Actions

**What I'm building on the side:**

[Folium EHR](https://github.com/pedropcamellon/folium) — an open-source electronic health record system I'm building from scratch as a personal engineering sandbox. FastAPI backend, React frontend, Temporal workflows, full observability stack, and a voice-note-to-clinical-summary pipeline powered by Whisper and LLMs. Not a product — a place to make real decisions and grow as an engineer, one PR at a time.

Healthcare has been an interest since my bachelor's project, where I used PyTorch to detect lung nodules in X-rays. Folium is the continuation of that thread — applying modern engineering to a domain that still needs it.

Personal projects let me explore alternative stacks or dive deeper into what I'm already using at work. Right now I'm exploring the full Grafana observability stack, formalizing my Azure knowledge with the AZ-104 Administrator cert as a step toward the DevOps Engineer cert, and staying hands-on with CI/CD pipelines and a security-first mindset.

**What I care about:**

Durable execution, platform engineering, and AI systems that solve real problems — not demos. I care about the craft: software architecture, beautiful code, and developer experience. I like working across the full reliability story — CI/CD, security, observability — and I'm drawn to where AI meets data: RAG, MLOps, vector databases, and the engineering discipline it takes to run these things in production. I believe in building in public, owning your tools, and writing about what you learn. I like supporting others and helping teammates grow.

Outside of work, I'm keeping a daily journal, figuring out the best way to grill on a Saturday afternoon with family, and staying curious about space exploration, music, history, renewable energy, and nature preservation.
